# 統合版：VWAP & Zig Zag & 単日四本値

TradingView 用 Pine Script v6 インジケーターです。VWAP・Zig Zag・単日四本値・前日四本値・**前日大引けVWAP**・移動平均線を1つのスクリプトに統合しています。

---

## 機能一覧

| 機能 | 概要 |
|------|------|
| **VWAP** | 当日始まりの出来高加重平均価格。標準偏差バンド付き |
| **移動平均線** | 5本・20本の単純移動平均線 |
| **単日四本値** | 指定した日付の始値・高値・安値・終値をチャート上に表示 |
| **前日四本値** | 前営業日の始値・高値・安値・終値を水平ラインで表示 |
| **前日大引けVWAP** | 前営業日の最終バー（例：15:30）時点のVWAPを水平ラインで翌日に表示 |
| **Zig Zag** | TradingView 公式ライブラリを使用したZig Zagライン |

## 動作要件

- Pine Script v6
- TradingView（出来高データが利用可能な銘柄・マーケット）

> 出来高データがない銘柄では `runtime.error` が発生します。

---

## 注意事項

- **単日四本値・前日四本値・前日大引けVWAP** は日足以上（日足・週足・月足）のチャートでは自動的に非表示になります。
- `max_lines_count=500`・`max_labels_count=500` を設定しています。長期間表示すると古いオブジェクトは自動削除されます。
- 前日四本値は `request.security` で日足データを取得しているため、リアルタイムバーでは確定値と異なる場合があります。
- 前日大引けVWAPは、日付切り替え時に **`vwapValue[1]`（前日最終バーのVWAP）** を確定値として採用します。

## 更新履歴

### feature/prev-day-vwap-line
- 前日最終バー（例：15:30）時点のVWAPをスナップショットし、翌日に水平ラインとして表示する「前日大引けVWAP」機能を追加（`vwapValue[1]` を日付切り替え時に確定）

### fix/table-global-and-prevhtf-trigger
- `request.security` の `[1]` オフセットを削除。`lookahead_off` では確定済み日足が基準となるため不要であり、残存により前々日の値が表示されていた問題を修正
- `var table info_table` をグローバルスコープへ移動。`barstate.islast` 時に `table.delete` → `table.new` でテーブル位置の変更を即時反映できるよう修正
- 前日四本値ラインの更新判定を `prevOpenHTF[1] != prevOpenHTF`（浮動小数点比較）から `timeframe.change("D")` に変更

### feature-refactor-globals
- `var table info_table` のグローバルスコープ移動（上記ブランチで改めて適用）
- 前日四本値ラインの更新判定を `timeframe.change("D")` に変更（上記ブランチで改めて適用）
- README に更新履歴セクションを追加

### feature-daily-ohlc-calculation
- 単日四本値のリセット・更新を対象日のバーのみに限定。全日で `sum_price_volume` / `sum_volume` が誤加算されていた問題を修正
- `daily_vwap_label` の `label.new` が未実装だったため追加
- `show_label_condition` から `is_target_date_bar` を除去し、過去日付でもラベルを表示できるよう修正
- `request.security` の `lookahead_on` を `lookahead_off` に修正
